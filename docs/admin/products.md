# Produits

La gestion des produits permet de créer et organiser votre catalogue.

## Liste des produits

Accédez à **Produits** dans la sidebar pour voir la liste de tous vos produits.

### Filtres disponibles
- **Recherche** : par nom ou SKU
- **Catégorie** : filtrer par catégorie
- **Statut** : Brouillon, Publié, Archivé
- **Stock** : En stock, Rupture, Stock bas

### Actions en lot
Sélectionnez plusieurs produits pour :
- **Publier** : passer en statut publié
- **Archiver** : retirer de la vente
- **Supprimer** : suppression définitive

## Créer un produit

1. Cliquez sur **Nouveau produit**
2. Remplissez les informations de base :
   - **Nom** : nom affiché sur la boutique
   - **Slug** : URL du produit (auto-généré)
   - **Description** : description détaillée (Markdown supporté)
   - **Catégorie** : catégorie principale
   - **Taux de TVA** : taux applicable

3. Enregistrez pour accéder aux options avancées

## Variantes

Les variantes permettent de gérer différentes déclinaisons d'un produit (taille, couleur, etc.).

### Créer des options
1. Dans l'onglet **Options**, cliquez sur **Ajouter une option**
2. Nommez l'option (ex: "Taille", "Couleur")
3. Ajoutez les valeurs (ex: "S", "M", "L")

### Générer les variantes
1. Cliquez sur **Générer les variantes**
2. Toutes les combinaisons sont créées automatiquement
3. Ajustez les prix et stocks pour chaque variante

### Informations par variante
- **SKU** : référence unique
- **Code-barres** : EAN/UPC
- **Prix HT** : prix de vente hors taxes
- **Prix barré** : ancien prix (promotions)
- **Prix de revient** : pour calcul de marge
- **Poids** : pour calcul des frais de port
- **Dimensions** : longueur, largeur, hauteur
- **Stock** : quantité disponible
- **Seuil d'alerte** : notification quand stock bas

## Images

### Ajouter des images
1. Cliquez sur la zone d'upload ou glissez-déposez
2. Les images sont ajoutées à la médiathèque automatiquement
3. Réordonnez par glisser-déposer

### Image principale
- La première image est l'image principale
- Cliquez sur l'étoile pour définir une autre image principale

### Images par variante
Associez des images spécifiques à chaque variante pour afficher la bonne image selon la sélection du client.

## Statuts

| Statut | Description |
|--------|-------------|
| **Brouillon** | Produit en cours d'édition, non visible |
| **Publié** | Visible sur la boutique |
| **Archivé** | Retiré de la vente, conservé en base |

## Bonnes pratiques

- Utilisez des **noms descriptifs** incluant les caractéristiques principales
- Ajoutez des **images de qualité** (minimum 800x800px)
- Renseignez le **poids et dimensions** pour des frais de port précis
- Définissez un **seuil d'alerte** adapté à votre rythme de vente
- Utilisez les **collections** pour regrouper des produits thématiques
