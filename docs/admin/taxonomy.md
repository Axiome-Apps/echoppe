# Catégories & Collections

Organisez votre catalogue avec des catégories et collections.

## Catégories

Les catégories permettent de classer vos produits de manière hiérarchique.

### Créer une catégorie

1. Allez dans **Organisation** > onglet **Catégories**
2. Cliquez sur **Nouvelle catégorie**
3. Remplissez :
   - **Nom** : nom affiché
   - **Slug** : URL de la catégorie
   - **Description** : texte descriptif
   - **Parent** : catégorie parente (optionnel)
   - **Image** : image de la catégorie

### Hiérarchie

- Glissez-déposez pour réordonner les catégories
- Imbriquez les catégories pour créer une arborescence
- Chaque produit appartient à une seule catégorie

### Visibilité

Désactivez la visibilité pour masquer une catégorie et tous ses produits de la boutique.

## Collections

Les collections regroupent des produits thématiques, indépendamment des catégories.

### Exemples d'utilisation
- Nouveautés
- Meilleures ventes
- Promotions
- Sélection saisonnière

### Créer une collection

1. Allez dans **Organisation** > onglet **Collections**
2. Cliquez sur **Nouvelle collection**
3. Remplissez :
   - **Nom** : nom de la collection
   - **Slug** : URL de la collection
   - **Description** : texte descriptif
   - **Image** : image de la collection

### Ajouter des produits

1. Ouvrez la collection
2. Cliquez sur **Ajouter des produits**
3. Sélectionnez les produits à inclure
4. Réordonnez par glisser-déposer

::: tip
Un produit peut appartenir à plusieurs collections, mais à une seule catégorie.
:::
