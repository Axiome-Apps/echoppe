# Médiathèque

Gérez vos images et fichiers.

## Interface

La médiathèque se compose de :
- **Sidebar gauche** : arborescence des dossiers
- **Zone principale** : fichiers du dossier sélectionné
- **Barre d'outils** : recherche, vue, upload

## Dossiers

### Créer un dossier
1. Cliquez sur **Nouveau dossier** dans la sidebar
2. Nommez le dossier
3. Sélectionnez le dossier parent (optionnel)

### Organiser
- Glissez-déposez les dossiers pour les réorganiser
- Renommez en double-cliquant sur le nom
- Supprimez via le menu contextuel (clic droit)

## Fichiers

### Upload

**Méthode 1 : Glisser-déposer**
- Glissez des fichiers depuis votre ordinateur vers la zone principale

**Méthode 2 : Bouton upload**
1. Cliquez sur **Upload**
2. Sélectionnez les fichiers
3. Les fichiers sont uploadés dans le dossier courant

### Formats supportés
- **Images** : JPG, PNG, GIF, WebP, SVG
- **Documents** : PDF
- **Taille max** : 10 Mo par fichier

### Actions sur les fichiers

**Sélection**
- Clic : sélectionner un fichier
- Ctrl/Cmd + clic : sélection multiple
- Shift + clic : sélection continue

**Menu contextuel (clic droit)**
- Renommer
- Déplacer vers un dossier
- Télécharger
- Supprimer

### Détails du fichier

Cliquez sur un fichier pour voir :
- Aperçu (images)
- Nom et titre
- Description et texte alternatif (SEO)
- Dimensions et poids
- Date d'upload
- URL du fichier

### Modifier les métadonnées

1. Ouvrez le détail du fichier
2. Modifiez :
   - **Titre** : nom affiché
   - **Description** : description longue
   - **Texte alternatif** : attribut alt pour le SEO
3. Enregistrez

## Vues

Basculez entre :
- **Grille** : aperçus en miniatures
- **Liste** : tableau avec détails

Ajustez la taille des miniatures avec le curseur.

## Recherche

Utilisez la barre de recherche pour trouver des fichiers par :
- Nom de fichier
- Titre
- Description
